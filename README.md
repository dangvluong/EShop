# EShop
A simple e-commerce web app created with ASP.NET Core MVC 5 and Dapper.

## Demo:
- http://dangvluongeshop.azurewebsites.net/
- Admin user: manager/manager
## Technology stack:
- ASP.NET Core MVC 5
- Dapper (connect to SQL Server)
